Introdução a Estruturas de Repetição

As estruturas de repetição while, do e for são usadas para executar um bloco de código várias vezes enquanto uma condição for verdadeira. Cada um dos três possui pequenas diferenças entre eles. While executa o teste de condição antes aplicar as instruções. Do aplica as instruções primeiro e no final executa o teste condicional. Já o for, muda como é escrita, ficando tudo no parênteses.

Estruturas de Repetição

As estruturas de repetição while e for são usadas para executar um bloco de código várias vezes enquanto uma condição for verdadeira.

Exemplo de While:


#include <stdio.h>

int main() {
    int i = 0;
    while (i < 5) {
        printf("%d\n", i);
        i++;
    }
    return 0;
}

Exemplo de For:


#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) {
        printf("%d\n", i);
    }
    return 0;
}

Exemplo de Do:

                    
                    #include <stdio.h>
                    
                    int main() {
                        int i = 0;
                        do {
                            printf("%d\n", i);  
                            i++;
                        } while (i < 5);  //A condicao e verificada apos a execucao do bloco
                        return 0;
                    }

Atribuição de valores em formas contraídas

Em laços de repetição, as formas contraídas de atribuições são essenciais para gerar atualização da variável e fazer com que a repetição siga até chegar a um certo valor e finalize o loop. Nos exemplos passados foi utilizado a forma “i++” que é igual a i=i+1, ou seja, atribui 1 a variável cada vez que passa no loop.

Exercícios

1. Escreva um programa que exiba os números de 1 a 10. Use o comando while.

                    

                
                    
                        #include <stdio.h>
                        int main() {
                            int i = 1;
                            
                            while (i <= 10) {
                                printf("%d\n", i);  
                                i++; 
                            }
                            getchar();
                            return 0;
                        }

2. Escreva um programa que peça ao usuário para adivinhar um número entre 1 e 10. O programa deve continuar pedindo até que o usuário acerte. Use o comando do-while.

                    

                    
                        
                            #include <stdio.h>
                            int main() {
                                int numeroSecreto = 6;  //Numero que o usuário deve adivinhar
                                int palpite;
                                
                                do {
                                    printf("Adivinhe o numero entre 1 e 10: ");
                                    scanf("%i", &palpite);
                                   fflush(stdin);
                                    if (palpite != numeroSecreto) {
                                        printf("Errado, tente novamente!\n");
                                    }
                                } while (palpite != numeroSecreto);
                                printf("Parabens, voce adivinhou o numero!\n");
                            
                            getchar();
                                return 0;
                            }

3. Escreva um programa que imprima a tabuada de um número informado pelo usuário, de 1 a 10. Use o comando for.

                    

                    
                        
                            #include <stdio.h>
                            int main() {
                                int num, i;
                                
                                printf("Digite um numero para ver sua tabuada: ");
                                scanf("%i", &num);
                                fflush(stdin);
                                
                                for (i = 1; i <= 10; i++) {
                                    printf("\n%i x %i = %i", num, i, num * i);
                                }
                                getchar();
                                return 0;
                            }
